# 背包问题核心笔记
---

## 1. 01背包 (每种物品仅有一件)

**核心定义：** 当前状态依赖于**上一层**的状态（确保每个物品最多只被选一次）。

图：  
![](/posts/cpp/assets/2026-03-08-18-35-24.png)

### 二维数组写法 (基础推导)
* **状态转移方程：** $dp[i][j] = \max(dp[i - 1][j], dp[i - 1][j - weight[i]] + value[i])$
* **遍历顺序：** 正序。因为有 `i` 和 `i-1` 的维度区分，不存在数据覆盖问题。

```cpp
// dp[i][j]：表示在只考虑前 i 个物品，且背包容量为 j 的情况下能装的最大价值
for (int i = 1; i < 物品.size(); ++i) {
    for (int j = 1; j <= 背包.size(); ++j) {
        if (j < weight[i]) {
            dp[i][j] = dp[i-1][j]; // 装不下，直接继承上一层
        } else {
            // max(不装, 装) -> 依赖上一层左边的数据: dp[i-1]
            dp[i][j] = max(dp[i-1][j], dp[i-1][j-weight[i]] + value[i]);
        }
    }
}
```

### 一维数组写法 (空间优化)  
* **状态转移方程：** $dp[j] = \max(dp[j], dp[j - weight[i]] + value[i])$  
* **遍历顺序：** 必须倒序！ 因为一维数组擦除了 i 的维度，倒序遍历才能保证在计算 dp[j] 时，它依赖的 dp[j - weight[i]] 还没有被当前物品 i 更新过（即本质上拿到的还是上一层的数据）。  
```cpp
for (int i = 1; i < 物品.size(); ++i) {
    // 倒序遍历容量，下限为当前物品的重量
    for (int j = 背包.size(); j >= weight[i]; --j) {
        dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);
    }
}
```

## 2. 完全背包 (每种物品有无限件)
**核心定义：** 当前状态依赖于当前层已经更新过的状态（允许同一个物品被多次放入）。  
### 二维数组写法 (基础推导)  
* **状态转移方程：** $dp[i][j] = \max(dp[i - 1][j], dp[i][j - weight[i]] + value[i])$  
* **注意细节：** max 的后半部分是 $dp[i]$ 而不是 $dp[i-1]$。
```cpp
for (int i = 1; i < 物品.size(); ++i) {
    for (int j = 1; j <= 背包.size(); ++j) {
        if (j < weight[i]) {
            dp[i][j] = dp[i - 1][j]; 
        } else {
            // max(不装, 装) -> 依赖当前层左边的数据: dp[i]
            dp[i][j] = max(dp[i - 1][j], dp[i][j - weight[i]] + value[i]);
        }
    }
}
```

### 一维数组写法 (空间优化)
* **状态转移方程：** 与01背包一维公式完全相同。  
* **遍历顺序：** 必须正序！ 正序遍历使得较小容量的状态先被当前物品更新，较大容量的状态在推导时，利用的就是已经装过当前物品的“脏数据”，从而实现了无限次添加。
```cpp
for (int i = 1; i < 物品.size(); ++i) {
    // 正序遍历容量，起点为当前物品的重量
    for (int j = weight[i]; j <= 背包.size(); ++j) {
        dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);
    }
}
```

## 3. 终极对比记忆模板 (一维空间优化版)
```cpp
for (int i = 1; i < 物品.size(); ++i) {
    
    // 【01背包】 - 倒序遍历 (防重复)
    for (int j = 背包.size(); j >= weight[i]; --j) {
        dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);
    }

    // ===================================================

    // 【完全背包】 - 正序遍历 (允重复)
    for (int j = weight[i]; j <= 背包.size(); ++j) {
        dp[j] = max(dp[j], dp[j - weight[i]] + value[i]);
    }
    
}
```
